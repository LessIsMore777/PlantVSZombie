# PlantVSZombie
植物大战僵尸 基于Cocos2D开发
##做游戏和放电影是一样的##
-荧幕 SurfaceView  界面展现  采用双缓冲机制，一个线程准备数据，另一个线程刷新界面
-放映机 SurfaceHolder 控制层
-工作人员 Surface 数据层

采用MVC模式
M--数据层 sqlite,SP,sdcard
V--xml
C--Activity

主要实现步骤：
1.展示logo
2.加载中的动画
3.点击开始按钮
4.菜单页面
5.地图移动
6.展现僵尸
7.已选植物框
8.选择植物框
9.选择/取消植物
10.开始战斗
11.地图移回去
12.展示开始安放植物的文字
13.安放植物（向日葵产生太阳）
14.僵尸出现
15.僵尸攻击植物
16.植物攻击僵尸
17.进度展示
18.背景音乐
19.金币展示

Tag:BattleLine是重点，各种重要的逻辑代码均在此处实现！！！
